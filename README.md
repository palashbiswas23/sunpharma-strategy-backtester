# 📈 SunPharma MA Crossover Strategy — Backtest

## Project Overview
A complete strategy backtester built on SunPharma (NSE) stock data 
from April 2023 to March 2025, using Moving Average Crossover signals.
Built entirely in Microsoft Excel and Power BI.

---

## Tools Used
- **Microsoft Excel** — Data cleaning, formula-based calculations
- **Power BI** — Interactive dashboard and DAX measures

---

## Strategy Logic
| Signal | Condition |
|--------|-----------|
| BUY  | 10-Day MA crosses ABOVE 30-Day MA |
| SELL | 10-Day MA crosses BELOW 30-Day MA |

## Project Structure
| File | Description |
|------|-------------|
| `SunPharma_Backtest.xlsx` | Raw data, MA calculations, signals, returns |
| `SunPharma_Backtest_Dashboard.pbix` | Power BI interactive dashboard |

## What I Learned
- How to build a trading strategy from scratch
- Moving Average Crossover logic and signals
- DAX measures in Power BI (Cumulative Return, Max Drawdown, Sharpe Ratio)
- Data analysis and visualization end to end
